# Application CRUD - Gestion des Utilisateurs et Championnats

Ce projet est une application web simple permettant de gérer des **utilisateurs** et des **championnats** via une interface d’administration. Il utilise un environnement **Docker (LAMP)** avec une base de données MySQL.

---

## 🚀 Fonctionnalités

- Création, modification et suppression d'utilisateurs
- Gestion des droits administrateurs (`is_admin`)
- Création, modification et suppression de championnats
- Interface d’administration sobre et épurée
- Authentification utilisateur
- Sécurité basique (rôles admin/utilisateur)

---

## 🛠️ Technologies utilisées

- HTML / CSS (design sobre : gris, blanc, vert)
- PHP (CRUD)
- MySQL (base de données)
- Docker / Docker Compose
- phpMyAdmin (accès à la base via le navigateur)
